Visualize your wall art from your WooCommerce shop. Plugin will help you embed the Wallnament widget

== Description ==

[Wallnament](https://www.wallnament.com) is a tool allowing your clients to visualize your artworks in their
own space using augmented reality technology.

The plugin simplifies and guides you through the widget installation process. The widget is a button directing your clients
to Wallnament mobile application.

== 3rd party and external services ==

This plugin uses one 3rd party service: [Wallnament](https://www.wallnament.com).
[Privacy policy](https://www.wallnament.com/privacy-policy) | [Terms of use](https://www.wallnament.com/terms-and-conditions)

Wallnament service usage:

* at the product page to inject and show Wallnament widget
* at the Wallnament admin page to show the widget preview and look configuration

== Installation ==

Wallnament plugin is a standard Wordpress plugin.

Please follow https://wordpress.org/support/article/managing-plugins/ installation guide.
